# System Patterns

## 架构

当前项目采用 Vite + React 的前端静态应用架构：

- `index.html`：Vite 应用挂载入口
- `src/main.jsx`：React 渲染入口
- `src/App.jsx`：落地页组件与内容数据
- `styles.css`：视觉系统、布局、响应式样式
- `resources/`：用户提供的物流图片素材

## 设计模式

- 单页落地页结构：导航、首屏、数据指标、服务、优势、资源展示、流程、联系 CTA、页脚
- React 函数组件拆分页面区块，便于后续维护和扩展
- 数组驱动服务、优势、流程等重复内容
- CSS 自定义属性集中管理品牌色、阴影、圆角与布局宽度

## 关键决策

- 使用 React + Vite，兼顾开发体验、组件化维护和静态部署便利性
- 图片直接从 `resources/` 引用，避免复制资源造成冗余
- 交互由 React 状态处理：移动导航、滚动导航状态、表单提交提示
